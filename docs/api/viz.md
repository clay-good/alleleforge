# Reproducible figures

`alleleforge.viz` renders the figures that ship in the README and the
[methods preprint](../paper/preprint.md). Every figure is computed from the
**weight-free, deterministic** pipeline — the same code paths the acceptance suite
and CRISPR-Bench exercise — so it regenerates byte-for-byte from config plus seed
(`python scripts/figures.py` or `make figures`) and is safe to commit and diff.

!!! note "No plotting dependency"
    The renderer is a small, hand-rolled SVG writer — the same
    no-heavy-dependency discipline as the [PDF report](report.md). There is no
    matplotlib/plotly in the dependency tree; output is a deterministic SVG string
    (no timestamps, no random ids, fixed number formatting).

## The figure set

| Figure | What it shows |
|---|---|
| `reference_bias` | A reference-only scan finds zero off-targets where the population-aware scan nominates one high-CFD site. |
| `conformal_coverage` | Raw vs recalibrated interval coverage at the nominal 80% / 90% targets. |
| `task_ece` | Per-task expected calibration error across the five CRISPR-Bench tasks. |
| `generalization_gap` | The cross-cell-type generalization gap per task (positive = worse). |

The per-task **values** are weight-free machinery checks (real numbers arrive with
the R1 real-weights integration); the renderer and the figure definitions are final.
